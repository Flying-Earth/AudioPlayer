# Audio Player

通过 Qt 制作的 Windows 端多媒体播放器，主要具备以下功能：

1. 添加多媒体文件，音视频播放分离。

2. 多媒体列表的基本播放功能及播放模式选择功能。

3. 通过 QSql 分别存储音频及视频列表。
